# MiTrabajoFinal

Proyecto Módulo 1 – Fundamentos

## Descripción:
Mi primera aplicación desarrollada en Streamlit que integra conceptos fundamentales de
programación como variables, condicionales, estructuras de datos, funciones,
programación funcional y POO.

El proyecto simula un sistema básico de gestión financiera, 
permitiendo verificar presupuestos, registrar actividades, 
calcular retornos esperados y modelar actividades como objetos.
